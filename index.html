Style Guide

    About

    The style guide is a CSS framework that forms the base of styling for all Fundly products (Fundly.com etc.)

    The Atomic CSS principles used in this framework were deeply inspired by this article.

    Repo

    https://github.com/fundly/style-guide

    Themes

    The style guide supports the concept of themes. Three colors can be set as part of the theme: Primary, AppRibbon, Footer

    Click one of the following themes to see its effect:

    Color

    Colors can be applied by the following selectors:

    • .f-fc-{KEY} - Font Color
    • .f-fc-hover-{KEY} - Font Color (hover)
    • .f-bg-{KEY} - Background Color
    • .f-bg-hover-{KEY} - Background Color (hover)
    • .f-bg-{KEY}-o-{OPACITY_KEY} - Background Color with opacity
    • .f-b-{KEY} - Border Color
    • .f-hr-{KEY} - Horizontal Rule Color

    Most core action buttons and highlights use the Primary color.

    Layout

    All layout spacing follows the convention: xsmall, small, medium, large, xlarge.

    Margin / Padding

    Margin: .f-m-{KEY} or .f-m-{SIDE}-{KEY}.

    Padding: .f-p-{KEY} or .f-p-{SIDE}-{KEY}.
